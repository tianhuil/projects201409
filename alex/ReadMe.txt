Source code is in "src/" the form of ipython notebooks.  Each .ipynb is
named according to the order in which it should be run.  Start with the
step0 (A and B) ipynb files to acquire the necessary data, and go from
there.  Any question, please contact aloosley@brown.edu

TO DEPLOY WEBSITE
1) index.html in the web folder is static
2) Simply run python simpleHTTPserver (python -m SimpleHTTPServer [port]) in the web folder
